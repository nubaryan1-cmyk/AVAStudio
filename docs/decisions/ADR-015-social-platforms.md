# ADR-015 — Провайдеро-независимая абстракция соцплатформ (SocialPlatform)

**Статус:** Принято · **Этап:** 12 · **Дата:** 2026-05

## Контекст
AVAStudio публикует контент в Instagram, TikTok, Reddit, Threads (далее +YouTube, X).
Платформы технически разнородны: одни дают официальный API (Reddit OAuth2, Threads
Meta Graph), другие требуют автоматизации браузера (Playwright) или действий на
облачном телефоне (anti-detect, через PhonePool). Прямая привязка постинг-логики к
каждой платформе/механизму = хрупкость и дубли при добавлении платформы.

## Решение
Единый интерфейс `SocialPlatform` (packages/shared/src/social/types.ts):
`login → postVideo/postPhoto/postCarousel → getStats`, плюс нормализованные типы
`Session`, `PostResult`, `AccountStats` и capabilities-флаги. Механизм реализации
(`api | browser | phone`) — деталь драйвера, скрытая за интерфейсом; вызывающий код
(scheduler 12.7, anti-ban 12.6) с механизмом не работает напрямую, лишь может задать
`preferMechanism`. Реестр (`createSocialRegistry`) маппит platform → драйвер.

## Маппинг платформ на механизмы (план)
- Instagram: основной `phone` (PhonePool, anti-detect), запасной `browser` (Playwright).
- TikTok: `phone` + `browser`.
- Reddit: `api` (официальный OAuth2).
- Threads: `api` (Meta Graph).

Один и тот же `SocialPlatform` покрывает все три механизма — драйвер инкапсулирует,
как именно выполняется действие; нормализованный `PostResult` единый.

## Фаза 1
Только mock-драйвер (`createMockPlatform`) — детерминированный, без сети/кредов;
проверяет контракт интерфейса и реестр. Реальные драйверы — TASK 12.4 (mock) и Фаза 2.

## Последствия
+ Новая платформа = новый драйвер + регистрация; постинг/scheduler не меняются.
+ Смена механизма (browser→phone) скрыта в драйвере.
+ Нормализованные результаты упрощают anti-ban/аналитику.
− Абстракция усредняет платформо-специфичные фичи (приемлемо; доступно через opts).

## Альтернативы
- Отдельный код на платформу без общего интерфейса — отклонено (дубли, хрупкость).
- Сторонний агрегатор постинга — отклонено (контроль, anti-ban, цена).
